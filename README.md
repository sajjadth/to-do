# to-do-app

simple to-do-app with User Authentication and MongoDb Atlas

### start server
```
cd server <br/>
npm start
```
### server development mode
```
cd server <br/>
npm run dev
```
### start clinet
```
cd client <br/>
npm start
```
### output clinet
```
cd client <br/>
npm run build
```
